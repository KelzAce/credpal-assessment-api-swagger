# CredPal Assessment API (Node.js + Express + TypeScript + MongoDB)

This repo is a complete CRUD API with JWT authentication and a protected resource (`transactions`).

## Quick start

1) Install deps

```bash
npm install
```

2) Create `.env`

```bash
cp .env.example .env
```

3) Start MongoDB (local) or set `MONGO_URI` to MongoDB Atlas.

4) Run dev server

```bash
npm run dev
```

API base: `http://localhost:4000/api/v1`

## API Docs (Swagger UI)

- Swagger UI: `http://localhost:4000/api/v1/docs`
- OpenAPI JSON: `http://localhost:4000/api/v1/docs.json`

In Swagger UI, click **Authorize** and paste the JWT you got from `/auth/login`.

## Endpoints

- `GET /health`
- `POST /auth/register`
- `POST /auth/login`
- `POST /transactions` (auth)
- `GET /transactions` (auth)
- `GET /transactions/:id` (auth)
- `PATCH /transactions/:id` (auth)
- `DELETE /transactions/:id` (auth)

## Postman

Import these files (generated by ChatGPT):
- `CredPal-Assessment.postman_collection.json`
- `CredPal-Assessment.postman_environment.json`

Run: Register → Login → Create Transaction → List/Get/Update/Delete.

## Notes

- All transaction operations are scoped to the authenticated user (multi-tenant safe).
- Uses `zod` for validation, `helmet` + `rate-limit` for basic hardening.

## Optional: run MongoDB with Docker

```bash
docker compose up -d
```
